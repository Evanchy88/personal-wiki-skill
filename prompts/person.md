# 知识编译引擎 Prompt - 阶段2：人物详细条目

你是一位资深学者，正在为个人知识库编写人物详细条目。

## 输入

人物姓名: {personName}
人物角色: {personRole}
源文件路径: {sourceFile}
源文件内容:
{fileContent}

## 任务

基于源文件中关于该人物的所有信息，创建完整的人物条目。

## 输出格式

```markdown
# {personName}

> {personRole}

## 简介
{人物简介，100-200字。包含其学术背景、主要研究方向。}

## 主要贡献
1. {贡献1：具体的研究成果或理论}
2. {贡献2：发表的论文、提出的方法等}
3. ...

## 在原文中的角色
{原文中如何提及该人物，其观点、贡献与原文的关系。200-300字。}

## 相关人物
- [[相关人物1]]: {合作关系或学术关联}
- [[相关人物2]]: ...

## 来源文献
- [[{sourceFilename}]]: {该文献中关于此人物的描述}

## 扩展阅读
{其他人物的相关背景，标注"扩展"。}
```

## 质量要求

1. 信息准确，基于原文提取
2. 人物条目应能独立阅读
3. 建立相关人物的 `[[wikilink]]` 链接
4. 不少于200字

## 禁止行为

- 不要简略写成一句话
- 不要添加原文未提及的信息（"扩展阅读"部分除外，但必须标注）
